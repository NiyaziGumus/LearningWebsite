Webentwicklung – Überblick

Worum geht's?

Webseiten bestehen aus HTML, CSS und optional JavaScript. Du brauchst ein Grundgerüst, gestaltest es mit Styles und achtest auf sichere Datenübertragung. Struktur und Design greifen ineinander.

Das Wichtigste in Kürze

  • HTML beschreibt Inhalt und Bedeutung mit Tags wie <header> oder <form>.
  • CSS sorgt für Farben, Layout, Abstände und responsive Anpassungen.
  • Formulare brauchen Labels, damit sie barrierefrei sind.
  • Responsive Design passt sich automatisch an kleine und große Displays an.
  • REST-Schnittstellen verbinden Frontend und Backend mit HTTP-Methoden.
  • HTTPS verschlüsselt den Datenverkehr zwischen Browser und Server.
  • SQL-Injection wird mit Prepared Statements verhindert.

Beispiel

<!DOCTYPE html>
<html lang="de">
<head><meta charset="utf-8"><title>Beispiel</title></head>
<body>Hallo Welt!</body>
</html>

Das Grundgerüst zeigt die wichtigsten Bereiche: Doctype, <head> und <body>.
