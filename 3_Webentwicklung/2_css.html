CSS gestalten

Mit CSS steuerst du Aussehen und Layout einer Webseite. Du kannst Farben, Schriftgrößen und Abstände definieren. Media Queries passen das Design an verschiedene Geräte an.

  • CSS lässt sich inline, intern im <style> oder extern per Datei einbinden.
  • Selektoren wie .klasse oder #id wählen Elemente aus.
  • Box-Modell: content, padding, border, margin.
  • Flexbox und Grid erleichtern moderne Layouts.
  • Media Queries reagieren auf Breite, z. B. @media (max-width: 600px).
  • Responsive Bilder nutzen max-width: 100% für Skalierung.
  • CSS-Dateien sollten organisiert und kommentiert sein.

body { font-family: Arial; margin: 0; }
.container { max-width: 900px; margin: 0 auto; }
@media (max-width: 600px) {
  .container { padding: 1rem; }
}

Das Layout zentriert den Inhalt auf großen Bildschirmen und sorgt auf kleinen Geräten für mehr Innenabstand.
