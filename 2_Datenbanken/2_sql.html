SQL-Abfragen schreiben

SQL ist die Sprache für relationale Datenbanken. Mit wenigen Schlüsselwörtern liest du Daten, filterst sie und kannst Auswertungen berechnen. Viele Datenbankwerkzeuge unterstützen denselben Standard.

  • SELECT wählt Spalten, FROM legt die Tabelle fest.
  • WHERE filtert Zeilen nach Bedingungen wie Preis > 100.
  • INNER JOIN kombiniert nur passende Zeilen, LEFT JOIN behält alle von links.
  • GROUP BY fasst Zeilen zusammen, z. B. pro Kunde.
  • Aggregatfunktionen wie COUNT, SUM, AVG, MAX berechnen Kennzahlen.
  • HAVING filtert Gruppen, ORDER BY sortiert das Ergebnis.
  • LIMIT schränkt die Anzahl der Zeilen ein.

SELECT k.name, COUNT(b.id) AS anzahl
FROM kunden k LEFT JOIN bestellungen b ON b.kunde_id = k.id
GROUP BY k.name HAVING COUNT(b.id) >= 0 ORDER BY anzahl DESC;

Der LEFT JOIN zeigt auch Kunden ohne Bestellung. HAVING filtert nach Gruppen, ORDER BY sortiert aufsteigend.
